# Lapidar Portal

Portal público gratuito de concursos literários, editais culturais, bolsas e chamadas abertas para escritores, poetas, roteiristas e autores independentes.

O portal reúne oportunidades nacionais e internacionais organizadas por categoria, facilitando o acesso de autores às principais portas de publicação, premiação e circulação cultural.

Projeto mantido por Site das Letras — Edições Literárias.
