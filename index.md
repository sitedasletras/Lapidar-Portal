# Lapidar Portal

Bem-vindo ao Lapidar Portal.

Este espaço reúne oportunidades reais para escritores e autores independentes:

- concursos literários
- editais culturais
- bolsas de criação
- chamadas de revistas
- oportunidades internacionais

O acesso é gratuito.
